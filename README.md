動作環境
--------

エレコマは次の環境で動作を確認しています。
（下記環境以外でも動作する可能性はあります）

- OS    : CentOS, FreeBSD
- DB    : PostgreSQL, MySQL, SQLite
- Ruby  : 2.2.0
- Rails : 4.2.1
